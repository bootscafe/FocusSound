Return To Nature

yokotsuka yuuya

02 Jul 2014
Amazon Apple

2013年のデビュー作で、繊細さと鋭さの混在した独自のサウンドで注目されたyokotsuka yuuyaの最新作が7/9に邂逅よりリリース。今作は一般流通のなかった自主盤として制作され、瞬く間に売り切れとなったものに書き下ろし曲を加えたセカンドフルアルバム。原点に戻り、過去の楽曲をアップデートさせたものも含め、自身の集大成とも言える内容となっている。「自然に帰る」をコンセプトに、あらゆる自然の情景を意識して制作、再構築された楽曲達は、持ち味とするノイズを用いたソリッドなビートメイキングと、ピアノを中心に奏でられる美麗なメロディが更に深化され、先端的でエモーショナルな美しき混沌を創造していく。
