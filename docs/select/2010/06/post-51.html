ギターを弾きながら聴きたい音楽

23 Jun 2010

前回を紹介しましたが色々な音楽を聴きながらギターを弾いてみてはいかがでしょう。ギターはたくさん聴いてコピーするのが唯一の上達の近道です。筆者の経験から、練習に最適なギターを弾きたくなるような音楽を紹介してみたいと思います。
