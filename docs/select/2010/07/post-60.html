早朝に聴くエレクトロニカ

26 Jul 2010

早朝に聴く音楽としてのエレクトロニカ作品を紹介します。ぼんやりとした意識をゆっくりと覚ませてくれるようなものから、さわやかな朝を演出してくれるものまで様々な観点からセレクトを行っています。

Read more
7fingers
Nils Frahm

さわやかなストリングスと繊細なエレクトロな音色が絶妙にブレンドされていて、とても心地よい作品です。控えめに鳴るピアノの音色も早朝にはぴったりで何度でも聴きたくなる作品です。

Amazon Apple
Read more
A Permanent Solution
Proem

重苦しくないライトなイメージのある作品。ゆったりとしたテンポの楽曲が多く、ウォーミングアップとして朝に聴くのに適した作品です。

Amazon Apple
Read more
Voiceband Jilt
Gregory Taylor

眠いときにゆっくりと目を覚ますのに聴くとよさそうな音楽です。ゆったりとした中にも躍動感があるため、ゆっくりと始動するには最適な音楽です。

Amazon Apple
Read more
EXP
Frank Bretschneider

生活の中に溶け込んでしまいそうな音がたくさん含まれている音楽です。じわじわと刺激を与えてくれるエレクトロニカ作品です。

Amazon Apple
Read more
Volume Objects
Autistici

朝、起きたすぐで何もやる気が起きないとき聴きたくなる作品です。適度に緊張感があるため、1日の中でやりたいことを考えるのにもよいでしょう。

Amazon Apple
Read more
Lupe Luep Peul Epul
Miki Yui

体調があまりよくない時や気分が乗らないときでも不思議と相性の合う作品。耳に痛い高音などはほとんど入っていないため、邪魔にならないBGMと言えるでしょう。

Amazon Apple
Read more
I Hold a Wish for You
aspidistrafly

絶対に朝に聴きたい。と思わせるような作品。アコースティックギターや女性ヴォーカルの優しい音色が早朝にとても合う美しい作品です。

Amazon Apple
Read more
The Elektronik Soft Heart of My Teknomodal Poet
Climnoizer

少し元気がある朝にはこの作品もよいです。ジャズやファンクを感じさせる少し変則的なエレクトニックサウンドです。朝食のBGMとして聴くのもよいと思います。

Amazon Apple
Read more
Pale Ravine
Deaf Center

旅行に行った時や、いつもと違う気分のときにかけたくなる音楽です。朝の景色を見ながら少し大きめの音で聴きたくなる躍動的な音楽です。
