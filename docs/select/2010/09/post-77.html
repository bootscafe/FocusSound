休日明けに聴きたいエレクトロニカ

23 Sep 2010

連休が明けてから体が疲れていたり、頭はスッキリしない方も多いのでないでしょうか。今回はゆっくりと体を休められて、頭をリラックスさせて仕事に集中できるようなエレクトロニカをセレクトしてみました。オフィスなどのBGMとしても活用できるものを紹介しています。
